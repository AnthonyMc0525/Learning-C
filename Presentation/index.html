Basic History

C was created in 1972 by a man named Dennis Ritchie.

C was made because the B language was not able to handle the new and upcoming challenges.

B was a slow language and could not take advantages of the PDP-11 computer

Uses for C

C either completely made, or largly made these operating systems:

  • Microsoft Windows
  • Linux
  • Mac

C also is used to write these database:

  • Oracle
  • MySQL
  • PostgreSQL

C is also really good at handling embedded systems like smart TV's and lots of "Internet of Things" (IoT)

General Information

C provides low level access to memory.

C is considered a Generic language. Meaning it is not only used for a few things. it is good at many different types of tasks

C's paradigms

  • Imperative - A programming paradigm that describes computation in terms of statements that change a program state
  • Procedural - A programming paradigm that describes, step by step, exactly the procedure that should be followed to solve a specific problem
  • C does not have built in OOP support

Well writen C code can be ran on many platforms and/or operating systems with little to no change to the source code

C is a compiled language

What is a compiler and how does it work

A compiler takes code written from a language and translates it into a language that the computer's proccesser can understand.

unlike an interpriter which scans the document line by line, a compiler scans the whole document before the program is run

Compilers generate error messages only after scanning the whole program. Which can make debugging comparatively hard.

compiling is generally faster than interpreting
